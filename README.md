# Seventeen — Simple Static Site

This repository contains a small static site for the Seventeen project.

Files
- [index.html](index.html) — Homepage
- [seventeen_member_hub.html](seventeen_member_hub.html) — Member hub page

Usage
- Open `index.html` or `seventeen_member_hub.html` in your browser.
- To serve locally (preferred), run:

```bash
python -m http.server 8000
# then open http://localhost:8000/
```

Notes
- Static HTML only — no build step required.
- Edit the HTML files directly to update content.
